A2 - Social Mashup

Assignment for Web Application Frameworks course

Purpose of this application

This assignment integrates 3 commonly used Web application APIs together as a tutorial for web framework applications. In the application I have used Facebooks API to retrieve current home location of users friends and then used the data to represent geographical distribution of his/her friends in a Google Maps application.

To ease up my implementation load, I have used several javascript technologies that enables me to do better web applications. For MVC model I have used Backbone, modularity was achieved via requireJS and jQuery was used to binder utility. For this being tutorial work on web application frameworks I decided to concentrate on trying out different javascript SDKs features. When implementing this I found out that for the Mashup application I didn't need to implement backend at all (no data persistence needed/desired).
