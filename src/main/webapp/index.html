Friends GeoLocation

A2 - Assignment for Web Application Frameworks course

Purpose of this application

This assignment integrates 3 commonly used Web application APIs together as a tutorial for web applications framework course. In the application I have used Facebooks API to retrieve current home location of users friends and then used the data to represent geographical distribution of his/her friends in a Google Maps canvas.

I have used several javascript technologies that enables me to do web applications easier. For MVC model I have used Backbone, modularity was achieved with requireJS, underscore was used for data parsing and jQuery was used as binder utility. For this being tutorial work on web application frameworks I decided to concentrate on trying out different features of SDKs that different vendors provided. When implementing this I found out that for this particular Mashup application I didn't need to implement backend at all (no data persistence needed/desired).
